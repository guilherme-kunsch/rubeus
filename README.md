# Projeto de Testes - QA Júnior

Este projeto foi desenvolvido como parte de um desafio prático para vaga de **QA Júnior**, com o objetivo de validar funcionalidades de uma aplicação web, identificar bugs e propor melhorias.

O foco principal foi aplicar conceitos fundamentais de testes de software e automação utilizando Cypress.

---

## Objetivo do Projeto

Realizar testes funcionais na aplicação web, validando:

- ✅ Funcionamento de botões
- ✅ Redirecionamentos de páginas
- ✅ Envio de formulário com dados válidos
- ✅ Validação de campos obrigatórios
- ✅ Mensagens de sucesso e erro
- ✅ Comportamentos inesperados
- ✅ Identificação de bugs e melhorias

---

## Abordagem de Testes

Durante o desenvolvimento foram aplicados os seguintes conceitos:

- ✔️ Testes positivos e negativos
- ✔️ Testes exploratórios
- ✔️ Organização de cenários de teste

Além da automação, foram analisados pontos de melhoria na experiência do usuário e inconsistências na interface.
